🧠 AI Mock Interview System
📌 Overview
The AI Mock Interview System is a web-based application that simulates real interview experiences. It helps users practice interviews, get AI-generated questions, and receive feedback to improve their performance.
🚀 Features
🎤 AI-generated interview questions (technical + HR)
⏱️ Timed mock interview sessions
📝 Answer evaluation and feedback
📊 Performance scoring system
📄 Resume-based question generation (optional)
💡 Improvement suggestions
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript (or React if used)
Backend: Python (Flask / Django / FastAPI)
AI Integration: OpenAI API / NLP models
Database: SQLite / MongoDB (optional)
Others: REST APIs, JSON

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/AI-Mock-Interview-System.git
cd AI-Mock-Interview-System
2. Create virtual environment
python -m venv venv
3. Activate environment
Windows:
venv\Scripts\activate
Mac/Linux:
source venv/bin/activate
4. Install dependencies
pip install -r requirements.txt
5. Run the project
python app.py
🧪 How It Works
User logs in / starts session
AI generates interview questions
User answers in real time
System evaluates answers
Feedback and score are displayed
